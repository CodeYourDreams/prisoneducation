Using String Indexing

Let's return to the aforementioned problem of extracting \"Bob\" from the variable boss = \"Billy Bob \". To start, let's begin by just extracting the 'B' from \"Bob\". When you want to extract an individual character from a string, you first name the string, followed by \"[]\". You would then place the index of the desired character inside the square brackets.


Example:

If I wanted to extract the second 'B' from boss, it would look like so:


1: boss = \"Billy Bob \"

2: c = boss[6]

3: print(c)


When run, the above program would display B on your screen, since the second 'B' has the index 6. Since we can concatenate two strings using the \"+\" operator, we could type the following code in order to print Bob onto our screen:


1: boss = \"Billy Bob \"

2: print(boss[6] + boss[7] + boss[8])


Since in computer science we want to condense our code as much as possible, Python allows us to extract multiple characters from a given string at a time as follows:


1: boss = \"Billy Bob\"

2: print(boss[6 : 9])


Here, \"Bob\" would still be displayed on our screen; however, the volume of code required is significantly reduced compared to the previous method. When we need to retrieve multiple characters from a given string, the familiar approach of naming the string followed by the square brackets remains unchanged. Instead of directly inputting the index of the desired character, a slight modification is applied. You begin by indicating the index from which the extraction should initiate-this index is included in the extraction. Following that, you specify the index of the character where the string's extraction concludes-this index is excluded. Essentially, the character located at the starting index will be included in the extraction, while the character at the final index will not be encompassed.
