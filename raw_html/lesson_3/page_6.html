What is a List?

Now that we've covered the string datatype, let's talk about another datatype used in Python that also uses indices; lists. Lists are an abstract form of data in Python that we use to store large or small groups of data. Just like a list in real life consists of a group of objects, a list in python consists of a group of data.


How Do We Create a List?

The first way to fill our list \"l\" with numbers one through five is to explicitly type in what integers you want to add to your list inside of the square brackets. The second way to add the numbers one through five would be as follows;


1: l = []

We assign the command \"append()\" to our list \"l\" by attaching the two via a period \".\". Since we print list \"l\" on line seven, we'd see [1, 2, 3, 4, 5] printed out onto our screen.


Note: In the above example, we stored the integer datatype inside our list, but don't be led astray! In Python, we can store any datatype inside of lists! This means we can have lists filled with strings, integers, Booleans, and even other lists!
