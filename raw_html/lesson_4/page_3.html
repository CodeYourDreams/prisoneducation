Dictionaries

Lists are a datatype that allow us to store and manipulate large amounts of data by assigning each piece of data an index number, which tells us how far that item is form the start of said list. What if we wanted to categorize our data differently? What if we sought to organize our data in a distinct manner? Imagine if, rather than relying on numerical indices, we attributed a unique key to each data element - a symbol that could be employed later in our code to retrieve specific pieces of information.

\n\n

This is the logic that leads us to dictionaries , the third datatype we will be covering in the course! To understand how a dictionary works, you should think back to how you would use a real life dictionary book. There, if you wanted the definition of a specific word, you'd use the word as a key to search through the book and find it's associated definition.

\n\n

The same is true for Python! As shown in the above image, we create a dictionary in python by typing out the name of the dictionary we want to create, followed by curly brackets (not square brackets, we use those for lists!). Notice how we fill dictionaries with key value pairs , where each key value pair is separated from the other inside the dictionary via commas.
