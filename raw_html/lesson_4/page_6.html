Tuples

Up to this point, you've explored the datatypes of strings, lists, and dictionaries. These datatypes collectively store collections of data and possess the capability to be modified after their creation. For instance, adjusting an element within a list or string is straightforward by referencing the index that needs editing, while the manipulation of a key-value pair in a dictionary is accomplished by a specific key.


But what if we wanted to create a datatype that can store large amounts of data that is unchangeable, meaning that once we store it we cannot change it later on? The answer is Tuples !

\n\n

Tuples are a data structure designed to store ordered collection of items. Unlike lists, tuples are immutable , meaning their elements cannot be changed once their defined. This property makes tuples particularly useful where data integrity are crucial. Tuples are created by enclosing comma separated values within parenthesis. Just like before, we use indices in order to call upon specific elements inside a tuple just like in lists or strings!
