How does going on the web work?

Our websites live on a server. Other people can talk to our server, and our server will send them back a website.

Flask! Flask is a neat Python library that lets us turn a computer into a server. We can make a MVC website using Flask along with a few other libraries.

  1. Flask is basically the controller!

  3. Flask lets us make unique and personalized websites easily
