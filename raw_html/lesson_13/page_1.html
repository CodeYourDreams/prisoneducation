Linking stuff with Flask: CSS

In Flask, you have a part where you write the words and set up the pictures - HTML. It's like the \"skeleton\" of your website. But for colors and designs, you use a different file, which in our case would be a CSS file like 'style.css'. You use little bits of code inside double curly brackets {{ }} to indicate that you're embedding python code in the HTML file to apply the styles from style.css.

{{ url_for(...) }} is a Flask template variable that generates the correct URL for the static resources. The 'static' argument in url_for() refers to the \"static\" folder in your Flask project where you typically store static assets like CSS, images, and JavaScript files.

static/style.css: This is the CSS file that you're linking to your HTML template.


Linking stuff with Flask: Nav Bar

Imagine you're building a website using Flask, which is like a special tool to make websites. And you want to have a cool navigation bar at the top of your web pages. This navigation bar helps people easily move around your website, like clicking buttons to go to different pages.

You have your main.py file, which is the main controller of the website. Then, on both the home.html and about.html pages, y ou have a special part called \"nav-bar\". This is where your navigation buttons will be.\nYou use {{url_for('home_fn')}} to create a link back to the home page. It's like making a signpost that says \"Go back home\".\nSimilarly, you use {{url_for('about_fn')}} to create a link to the about page.

So, when someone goes to your website's home page or the about page, they see a nice navigation bar at the top. This helps them easily move between different parts of your website. It's like having signposts to guide them, and Flask helps you set up these signposts to make your website user-friendly!
